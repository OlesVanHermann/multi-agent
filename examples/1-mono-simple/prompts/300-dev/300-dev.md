## Methodology

## Quand tu reçois "go"
1. Compter les PR-SPEC pending
2. Si count = 0 → terminé, signaler
3. Sinon → prendre le premier et le traiter
4. Après traitement → s'auto-injecter "go" : `$BASE/scripts/send.sh 300 "go"`

## Traitement d'un PR-SPEC-300-{ID}
1. LIRE le PR : `cat $BASE/pool-requests/pending/PR-SPEC-300-{ID}.md`
2. MOVE PR vers assigned : `git mv pending/PR-SPEC-300-{ID}.md assigned/`
3. LIRE le SPEC référencé
4. IMPLÉMENTER la fonction
5. COMMIT le code : `git add -A && git commit -m "feat: implement {function}"`
6. MOVE PR vers done : `git mv assigned/PR-SPEC-300-{ID}.md done/`
7. NOTIFIER via Redis :
   ```bash
   $BASE/scripts/send.sh 400 "commit: $HASH"
   $BASE/scripts/send.sh 500 "PR-SPEC-300-{ID}"
   ```
