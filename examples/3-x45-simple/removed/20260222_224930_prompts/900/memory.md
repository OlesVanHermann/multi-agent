# 900 — Memory

## Contexte projet
Projet d'analyse de données de ventes. Un fichier CSV contenant les ventes
mensuelles 2025 par produit et région. L'objectif est de produire un rapport
structuré avec tendances et recommandations.

## Données brutes
- Type : CSV (séparateur virgule)
- Fichier : `project/raw/ventes-2025.csv`
- Colonnes : date, produit, region, quantite, prix_unitaire, total

## Output attendu
Un rapport markdown dans `project/pipeline/341-output/rapport-ventes.md`
avec : résumé exécutif, top produits, tendances par région, recommandations.
