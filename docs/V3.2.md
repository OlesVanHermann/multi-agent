# Multi-Agent v3.2 — preuve des boucles auto-améliorantes

## Évolution v3.2.X des prompts

La ligne v3.2.X introduit une écriture résultat-first : 70 % résultat métier,
20 % vérification, 10 % processus. Les créateurs 150/160/170, les templates et
les exemples suivent ce contrat. `upgrade.sh` migre aussi les `system.md` projet
existants de façon idempotente et récupérable.

Voir [HOW TO WRITE AND REWRITE PROMPTS](HOW_TO_WRITE_AND_REWRITE_PROMPTS.md).

La version 3.2 étend la preuve du bridge aux modes x45/z21 et fournit des
observateurs paramétriques. Contradictor occupe un slot `NNN-2XX` ; l'Ami
expérimental de l'étude n'est pas retenu dans la topologie livrée.

Dans x45/z21, Contradictor est un satellite local du triangle et son contrôle apparaît
dans le diagramme de ce triangle, jamais dans un panneau global. Le format mono
v3.2 est une paire : principal `3XX-1XX` et Contradictor `3XX-2XX`.

## Installation nouvelle ou existante

`upgrade.sh` met à jour le framework et les créateurs 150/160/170, réécrit les
contrats des agents projet, transforme les mono historiques en paires et ajoute
un Contradictor aux x45/z21 qui n'en ont pas. Aucun service n'est démarré. Le
scaffolding manuel reste disponible pour une création isolée ou un cas `MANUAL` :

Depuis v3.1.17 ou une version antérieure, exécuter `upgrade.sh v3.2.0` deux
fois : la première passe installe le nouvel upgrade, la seconde migre les agents.
À l'issue des deux passes, ne démarrer ni l'infrastructure ni les agents : le
démarrage est une opération opérateur distincte de l'upgrade.

```bash
python3 scripts/scaffold-observers.py 427 \
  --directory-name 427-mon-projet \
  --contradictor-suffix 217 --contradictor-login login3a
```

Le scaffolder archive tout homonyme dans `removed/`, crée prompts, liens de
configuration et workflow matérialisé. Il ne démarre aucun service et
n'authentifie aucun compte.

Pour convertir le mono initial créé par 150 en mono-pair :

```bash
python3 scripts/scaffold-mono-pair.py 345 \
  --directory-name 345-mon-agent \
  --main-suffix 145 --contradictor-suffix 245
```
