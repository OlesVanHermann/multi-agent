## Anomalies connues

### messaging
- `0-manage-*` dans plan-DONE : dossiers crawles bruts, non categorises sous une lettre → a migrer vers C-notifications ou I-administration
- `A_messaging_core` dans plan-DONE : doublon de `A-messaging-core` avec underscore → doublon a supprimer apres verification
- `G-apps-0-{app-id}-{app-name}` : granularite trop fine (1 app = 1 dossier), devrait etre sous `G-apps/0-{subcategory}/`

### mail
- Lettre B absente : le service mail n'a pas de categorie B (normalement search/organisation)
- Lettre E-F absentes : features avancees non documentees ou non planifiees

### edition
- `H-mcp` : lettre H utilisee pour MCP (integration), hors convention (devrait etre G ou I)
- Lettres F-G absentes

### desktop-android / desktop-windows
- Pattern numerique (1-, 2-) au lieu d'alphabetique (A-, B-)
- Coherent entre les deux services — ne pas changer sans decision du Master
