# 011-711 Methodology — Curator

## Synthèse de spec
1. Lire `plan-DOING/{TASK_ID}.md` intégralement
2. Extraire : fichier source, type de transformation, critères de succès
3. Si info manquante : chercher dans `plan-DONE/` les tâches similaires pour inférer le pattern
4. Réécrire `011-011-memory.md` en remplissant tous les champs

## Format memory.md
Le fichier doit permettre au Dev de travailler sans lire la spec directement.
Informations obligatoires :
- Chemin absolu du fichier source
- Description précise de la transformation en 1-2 phrases
- Critères de succès vérifiables (pas vagues)
