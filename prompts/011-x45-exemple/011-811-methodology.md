# 011-811 Methodology — Coach

## Arbre de décision

| Score | Problème | Action |
|-------|----------|--------|
| < 60 | Contenu incorrect | Clarifier la transformation dans methodology |
| 60-79 | Format / structure | Ajouter des exemples concrets dans la checklist |
| 80-97 | Détails manquants | Ajouter une règle spécifique dans "Points d'attention" |
| ≥ 98 | — | Ne pas appeler le Coach |

## Principe
Modifier **uniquement** la section qui cause le score insuffisant.
Ne pas réécrire la methodology entière — correction chirurgicale.
