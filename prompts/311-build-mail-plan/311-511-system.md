## Methode d'evaluation

### Checks automatiques
| Check | Critere |
|-------|---------|
| Presence du fichier `{Cat}_{num}_{name}.py` | C5 (Executabilite) |
| `import pytest` present | C5 |
| Au moins 8 fonctions `def test_` ou methodes dans classes `Test*` | C1 |
| `import imaplib` ou `import socket` present | C3 |
| Assertions non triviales (pas juste `assert True`) | C2 |
| `pytest.skip(...)` pour connexion non disponible | C3 |
| Tests pour les chemins d'erreur (except, status != 'OK') | C4 |
| Fixtures ou setup/teardown present | C3 |

### Grille de scoring (5 criteres)
| # | Critere | Poids | 100% si... | 0% si... |
|---|---------|-------|-----------|----------|
| C1 | **Coverage** | 30% | Tous les comportements cles de 311-memory.md section 2 sont testes | Moins de 3 tests |
| C2 | **Assertions** | 25% | Assertions verifient reponses IMAP reelles (status 'OK', contenu, structure) | Assertions triviales ou absentes |
| C3 | **Protocol** | 20% | imaplib.IMAP4_SSL ou socket raw utilises correctement, fixtures propres | Aucune connexion IMAP |
| C4 | **Edge cases** | 15% | Chemins d'erreur, inputs invalides, timeouts testes | Uniquement happy path |
| C5 | **Executabilite** | 10% | Syntaxe valide, imports existants, pytest compatible | Erreur de syntaxe |

### Score = somme(critere × poids), arrondi a l'entier

### Penalites automatiques
- Pas de fichier dans pipeline/311-output/ → score 0
- `assert True` sans logique → -10 sur C2
- Aucun import imaplib/socket → -15 sur C3
- Moins de 8 tests → proportionnel sur C1
