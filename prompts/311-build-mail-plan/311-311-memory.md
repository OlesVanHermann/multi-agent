1. **Hero** : titre + sous-titre (win clients, meet requirements, protect data) + 2 CTAs + trust banner (50k+ businesses, 100M people)
2. **What is ISO 27001** : intro + 5 benefits
   - Expand client base, Build trust instantly, Access global markets, Protect against breaches, Strengthen compliance (GDPR, HIPAA)
3. **ISO 27001 vs SOC 2 Comparison Table** (7 rows) :
   - Target market: International vs North America
   - Validity: 3yr annual audits vs 1yr
   - Timeline: ~10mo vs ~4-7mo
   - Flexibility: Fixed scope vs Flexible
   - Documentation: Comprehensive vs Varies
   - Cost: $10k-25k vs $10k-60k
   - Auditor/Accreditation: ANAB vs AICPA
4. **Requirements** (7 clauses, 4-10) :
   - C4: ISMS scope, C5: Leadership commitment, C6: Security objectives, C7: Resource plan, C8: Operational plan, C9: Success metrics, C10: Non-conformity logging
5. **Certification Process** (4 steps) : Assess security → Implement controls → Document practices → Official audit
6. **Proton Tools** (4 products for ISO compliance) :
   - Drive (secure file sharing, access mgmt, activity logging)
   - Mail (encrypted email, enforced 2FA, zero-access)
   - VPN (network security, access segmentation, no-log)
   - Pass (password manager, 2FA integration, proactive security)
7. **Trust Badges** (4) : ISO 27001 certified, Always transparent (open source), Built for privacy (E2E), Swiss privacy laws
8. **FAQ** (6 questions) : What is it, Cost ($10k-25k), How to get certified (4 steps, 6-12mo), How to maintain, Why important, Is it worth it
