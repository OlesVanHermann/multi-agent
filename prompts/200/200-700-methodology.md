# 200-700 — Methodology

## Étapes
1. Lire `prompts/200/200-system.md` pour les besoins de 200
2. Scanner `project/raw/` : lister fichiers, types, tailles
3. Scanner `project/clean/` : fichiers déjà nettoyés
4. Synthétiser :
   - Types de fichiers par catégorie (CSV, HTML, PDF, JSON)
   - Fichiers restant à nettoyer
   - Règles spécifiques au projet
5. Vérifier < 2000 tokens
6. Écrire dans `prompts/200/200-memory.md`
